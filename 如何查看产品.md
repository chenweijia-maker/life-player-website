# 如何查看「人生玩家」产品

## 🌐 方式一：Web 浏览器查看（推荐，最简单）

### 步骤：

1. **启动后端服务器**
   
   打开终端，在项目根目录运行：
   ```bash
   npm start
   ```
   
   你会看到类似输出：
   ```
   后端服务器运行在 http://localhost:4000
   ```

2. **打开前端页面**
   
   有两种方式：
   
   **方式 A：使用本地服务器（推荐）**
   ```bash
   # 新开一个终端窗口，在项目根目录运行
   npx serve frontend
   ```
   
   然后在浏览器访问：`http://localhost:3000`
   
   **方式 B：直接打开文件**
   - 找到 `frontend/index.html` 文件
   - 双击打开（会在默认浏览器中打开）
   - ⚠️ 注意：直接打开文件可能会有 API 请求失败的问题，建议使用方式 A

3. **开始使用**
   - 在登录页面输入手机号
   - 验证码固定为：`123456`（开发环境）
   - 登录后即可使用所有功能

## 📱 方式二：iOS APP 查看（需要 macOS）

### 前置要求：
- macOS 系统
- Xcode（从 App Store 安装）
- CocoaPods（`sudo gem install cocoapods`）

### 步骤：

1. **安装依赖**
   ```bash
   npm install
   ```

2. **构建前端资源**
   ```bash
   npm run build
   ```

3. **初始化 iOS 项目**（首次运行）
   ```bash
   npm run cap:add:ios
   npm run cap:sync
   ```

4. **在 Xcode 中打开**
   ```bash
   npm run cap:open:ios
   ```

5. **配置和运行**
   - 在 Xcode 中选择开发团队
   - 选择模拟器或真机
   - 点击运行按钮（▶️）

6. **配置后端服务器地址**
   
   编辑 `capacitor.config.js`，将 `server.url` 改为你的 Mac IP 地址：
   ```javascript
   server: {
     url: 'http://192.168.1.100:4000',  // 替换为你的 Mac IP
     cleartext: true
   }
   ```
   
   查找 Mac IP：
   ```bash
   ifconfig | grep "inet " | grep -v 127.0.0.1
   ```

## 🔧 常见问题

### Q: 后端启动失败？
**A:** 检查：
- Node.js 是否已安装：`node --version`
- 依赖是否安装：`npm install`
- 端口 4000 是否被占用

### Q: 前端页面空白或报错？
**A:** 检查：
- 后端是否正在运行（`npm start`）
- 浏览器控制台（F12）是否有错误信息
- 是否使用了本地服务器（`npx serve frontend`）

### Q: API 请求失败（404、CORS 错误）？
**A:** 
- 确保后端服务器正在运行
- 检查 `frontend/index.html` 中的 API 地址是否正确
- 如果直接打开 HTML 文件，请改用本地服务器

### Q: iOS APP 无法连接后端？
**A:**
- iOS 设备不能使用 `localhost`，必须使用 Mac 的局域网 IP
- 确保 Mac 和 iOS 设备在同一网络
- 检查防火墙设置
