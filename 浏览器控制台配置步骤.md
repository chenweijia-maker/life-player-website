# 浏览器控制台配置 API 地址 - 详细步骤

## 📋 完整操作步骤

### 第一步：打开浏览器开发者工具

**方法一：快捷键（推荐）**
- **Windows/Linux**：按 `F12` 键
- **Mac**：按 `Command + Option + I`（⌘ + ⌥ + I）

**方法二：右键菜单**
1. 在网页上**右键点击**任意位置
2. 选择 **"检查"** 或 **"Inspect"** 或 **"审查元素"**

**方法三：浏览器菜单**
- **Chrome/Edge**：点击右上角三个点 `⋮` → 更多工具 → 开发者工具
- **Firefox**：点击右上角三条线 `☰` → 更多工具 → Web 开发者工具
- **Safari**：需要先在"偏好设置"中启用"开发"菜单

### 第二步：切换到 Console（控制台）标签

打开开发者工具后，你会看到顶部有几个标签：
- **Elements**（元素）
- **Console**（控制台）← **点击这个**
- **Network**（网络）
- **Sources**（源代码）
- 等等...

点击 **Console** 标签页。

### 第三步：在控制台输入代码

在控制台底部的输入框中（通常显示 `>` 或 `▷`），输入以下代码：

```javascript
localStorage.setItem('api_base_url', 'http://your-server.com:4000');
```

**⚠️ 重要：** 请将 `http://your-server.com:4000` 替换为你的实际后端服务器地址！

**示例：**
- 如果后端在局域网：`http://192.168.1.100:4000`
- 如果后端在公网：`http://123.456.789.0:4000`
- 如果有域名：`https://api.your-domain.com`

**完整示例：**
```javascript
localStorage.setItem('api_base_url', 'http://192.168.1.100:4000');
```

### 第四步：按回车执行

输入完代码后，按键盘上的 **Enter（回车）** 键执行。

如果设置成功，控制台不会显示任何错误（或者显示 `undefined`，这是正常的）。

### 第五步：验证设置是否成功

在控制台输入以下代码检查：

```javascript
localStorage.getItem('api_base_url');
```

按回车后，应该会显示你刚才设置的地址，例如：
```
"http://192.168.1.100:4000"
```

### 第六步：刷新页面

按 `F5` 或点击浏览器的刷新按钮，页面会使用新配置的 API 地址。

## 🎯 完整操作示例

假设你的后端服务器地址是 `http://192.168.1.100:4000`，操作如下：

1. **打开页面**：访问 `http://localhost:3000`（或你的前端地址）
2. **按 F12**：打开开发者工具
3. **点击 Console 标签**
4. **输入代码**：
   ```javascript
   localStorage.setItem('api_base_url', 'http://192.168.1.100:4000');
   ```
5. **按回车**
6. **验证**（可选）：
   ```javascript
   localStorage.getItem('api_base_url');
   ```
   应该显示：`"http://192.168.1.100:4000"`
7. **刷新页面**：按 F5

## ⚠️ 常见问题

### Q1: 控制台显示红色错误？
**A:** 检查：
- 地址是否正确（包含 `http://` 或 `https://`）
- 引号是否配对（前后都是单引号或双引号）
- 是否有拼写错误

### Q2: 刷新后还是连接 localhost？
**A:** 检查：
- 是否真的执行了代码（按了回车）
- 是否在正确的页面（前端页面，不是后端页面）
- 尝试清除浏览器缓存后重试

### Q3: 如何清除已设置的地址？
**A:** 在控制台输入：
```javascript
localStorage.removeItem('api_base_url');
```

### Q4: 如何查看所有 localStorage 数据？
**A:** 在控制台输入：
```javascript
console.log(localStorage);
```

## 💡 小技巧

1. **自动补全**：输入 `localStorage.` 后，浏览器会自动提示可用方法
2. **历史记录**：按键盘上下箭头可以查看之前输入的命令
3. **多行输入**：按 `Shift + Enter` 可以换行，输入多行代码
4. **清空控制台**：点击控制台左上角的 🚫 图标，或输入 `console.clear()`

## 🔄 其他配置方法

如果觉得控制台方法麻烦，也可以：

**方法一：URL 参数（更简单）**
直接在浏览器地址栏输入：
```
http://localhost:3000/?api=http://192.168.1.100:4000
```

**方法二：修改代码**
直接编辑 `frontend/index.html` 文件，修改默认地址。
